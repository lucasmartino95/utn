# UML

Orden de elementos al diagramas una clase:

```
Nombre de la clase

Atributos  

Constructores    
Getters y setters  
Métodos de instancia  
Elementos estáticos  (también hay que subrayarlos con _ al inicio y final del elemento)
```

Símbolos:

```
Público: +  
Privado: -  
Protegido: *  
De paquete: no se representa
```
