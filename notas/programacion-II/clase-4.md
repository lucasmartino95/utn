# Sobrecargas

Definir múltiples miembros con el mismo nombre, pero con diferente 
listas de parámetros.

En **Java** se pueden sobrecargar los métodos y el constructor.

Los **métodos sobrecargados** deberían **hacer lo mismo** con diferentes tipos de datos.

# La clase StringBuilder

Cambia el contenido de un String **sin crear nuevos objetos**, por esto es más eficiente que usar un String literal. También cuenta con **métodos para poder manipular** con más herramientas a los Strings.

# Diferencia entre equals y ==

Equals es para Strings y **==** es para datos primitivos o referencias en el stack.

# Enums

Son útiles para **representar un conjunto fijo de valores conocidos**, como los días de la semana, los colores, los estados de un proceso, etc.

Con los enums una validación menos para hacer.
